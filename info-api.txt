dijalankan oleh program lain.

Didalam API ada Webservice didalam Webservice ada REST API
Fungsi API untuk membuat webservice(data) menjadi bisa diakses oleh banyak platform
JSON adalah format standar data

RestAPI adalah standar untuk membuat sebuah API(Webservice )
Https bisa digunakan untuk bertukar data menggunakan url/internet

 Same-origin policy adalah aturan/kebijakan untuk mengakses dan menampilkan data/script dll dari tempat lain diluar domainnya(online)

 CORS(Cross Origin Resource Sharing) adalah cara konfigurasi yang membuat kita dapat mengakses sebuah resouce dari domain lain,caranya dengan membuat server yang memperbolehkan/mengijinkan adanya akses data dari luar domain,
 memodifikasi Access-Control-Allow-Origin dengan membuat file .htaccess
